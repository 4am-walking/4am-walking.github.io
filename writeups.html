What are boxes and CTFs?

Boxes: Hack the Box is a platform hosting many different virtual machines or "boxes". These boxes each have different vulnerabilities and difficulty levels associated with finding these vulnerabilities. For example, an easy box will usually have more common and fewer vulnerabilities than a medium box. The goal of these boxes is to gain root/administrator access through exploiting vulnerabilities. Hack the Box isn't the only platform that hosts these types of boxes, however, it is the platform I love using the most.

CTFs: CTF or Capture the Flag is a general term referring to any challenge in which the user has to exploit some measure to get a "flag". These are usually more modular that full boxes and can challenge you on aspects taht don't really fit into boxes too well. For example, Hack the Box hosts many CTFs involving things like hardware hacking, game hacking, crytography, etc. The flag that you're after is acquired after you have found the vulnerability or secret. On a platform like Hack the Box, most CTF flags look similar to this: HTB{0s1n7_15_C001}.
